NFrostN

NFrostN is a particle diffusion algorithm. the logic of the particle diffusion generates an output picture that looks like frost on a windshield. This project was made for Dr. Ackley's class at UNM. This is not an orignal idea of mine, but the code is all mine. The project was originally built in Java, but I have ported it to Processing(Java version). I may bring it to JavaScript later to embed it in this webpage. For now check out the GitHub Repo and the Medium article for more details.

A Simple World

Here we have another application with inspiration from Dr. David Ackley. This is an ode to his artificial life video. I would go watch the video right now! CLICK HERE! Have fun getting lost in his teachings. This program is very simple version. It is a SimpleWorld :] the squares are plants that grow to the the cells around it. Once it expands and grows it because stale. It will no longer grow. There is also a Plant Eater. You can tell because they are circles and they look for plants to eat. They replicate after they eat enough food or they die if they wonder around enough without eating. They both co-exist in a simple world and live forever. I am sure you may want to read more on the Medium Article. If you want to see the code check out the Github repo.
